SVG: Scalable Vector Graphics

  • Graphics file format — .svg
  • Resolution independent
  • Created with programs like Illustrator and Sketch
  • May contain vector graphic shapes, bitmap (raster) images and text
<svg viewbox="0 0 200 100">
  <!-- Rectangle -->
  <rect width="100%" height="100%" fill="orange" />
  <!-- Circle -->
  <circle cx="150" cy="100" r="80" fill="blue" />
  <!-- Text -->
  <text x="150" y="115" font-size="60" fill="white">SVG</text>
</svg>

SVGs are XML based

Made up of tags, much like HTML, which define the 'structure' of the document.

Embedding SVG 'inline' in HTML

We can put the SVG markup directly into our website as an HTML element, and it will render.


<body>

  <!-- HTML -->
  <div>Your SVG can sit in your HTML!</div>

  <!-- SVG -->
  <svg viewbox="0 0 200 100" width="200" height="100">
    <!-- Rectangle -->
    <rect width="100%" height="100%" fill="orange" />
    <!-- Circle -->
    <circle cx="150" cy="100" r="80" fill="blue" />
    <!-- Text -->
    <text x="150" y="125" font-size="60" fill="white">SVG</text>
  </svg>

</body>

The viewBox attribute defines a grid

<svg viewBox="0 0 10 10">
  <!-- Elements would love to go here -->
</svg>

And then we can draw elements according to that grid

<svg viewBox="0 0 10 10">
  <polyline points="2,1 8,5 2,9 "/>
  <line x1="2" y1="5" x2="8" y2="5"/>
</svg>

Then you can easily add CSS to style and animate them!

svg .star {
  animation: rotate 10s infinite linear;
}

@keyframes rotate {
  0% { transform: rotate(0deg); }
  100% { transform: rotate(360deg); }
}

Animating lines using stroke-dasharray and stroke-dashoffset

We can use stroke-dasharray to define a 'dash' of stroke which is the length of the entire path.

You can find the length of an SVG path using Javascript

let path = document.querySelector('.star');
let length = path.getTotalLength();

We transition the stroke-dashoffset so the line is drawn back onto the path.

.star {
  stroke-dasharray: 390.317626953125;
  animation: draw 5s linear infinite;
}

@keyframes draw {
  0% {
    stroke-dashoffset: 390.317626953125;
  }
  100% {
    stroke-dashoffset: 0;
  }
}

Tutorial Time #1

Making An Animated Hamburger Menu Icon With SVG + CSS

1. We start by creating the 'rails' on which our lines will 'travel'.

2. We then define the stroke-dasharray and stroke-dashoffset to create the hamburger icon.

.top, .bottom {
  stroke-dasharray: 5.92 17;
  stroke-dashoffset: -10.4;
}

.cross {
  stroke-dashoffset: 5.9;
}

3. Then we move the top and bottom lines in using translate

/* Move top and bottom lines in to form a square */
.top {
  transform: translateY(1.5px);
}
.bottom {
  transform: translateY(-1.5px);
}

4. Finally, we work out the 'active' state, where the lines form the cross.

.active .top,
.active .bottom {
  stroke-dasharray: 8.3;
  stroke-dashoffset: 0;
  transform: translateY(0px);
}

.active .cross {
  stroke-dashoffset: 0;
}
.line {
  transition: 0.25s stroke-dasharray ease 0s,
  0.25s stroke-dashoffset ease 0s,
  0.25s transform ease 0.25s;
}

.active .line {
  transition: 0.25s stroke-dasharray ease 0.25s,
  0.25s stroke-dashoffset ease 0.25s,
  0.25s transform ease 0s;
}
